# Archived Status

The C# watcher is archived.

Reason:

- The Python implementation is now the single maintained production app.
- Keeping two watchers in parallel would double the maintenance cost without enough practical benefit.

What remains useful here:

- runtime and packaging experiments
- WinForms desktop prototype
- groundwork for a possible future Avalonia shell

Current rule:

- do not add routine feature work here unless the project is explicitly reactivated
